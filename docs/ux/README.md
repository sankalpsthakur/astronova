## Design System

Astronova uses a cosmic-themed design system:
- **Colors**: Gold, brass, copper accents on dark space backgrounds
- **Typography**: San Francisco with custom cosmic type scale
- **Animations**: Smooth, celestial motion patterns
- **Components**: Reusable cosmic-styled UI components

**Design System Files**: `client/AstronovaApp/Cosmic*.swift`
